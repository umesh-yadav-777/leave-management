# Employee Leave Management System (Frontend)

## 📌 Project Overview
This project is a React.js-based frontend application for managing employee leave requests. It provides a user-friendly interface for employees and administrators to handle leave workflows efficiently.

## 🚀 Features
- Employee login & registration
- Apply for leave
- View leave history and status
- User dashboard
- Admin dashboard
- Manage users and leave requests

## 🛠 Tech Stack
- React.js
- JavaScript
- HTML & CSS
- Bootstrap

## 📂 Project Structure
- Components: Navbar, Sidebar, Layouts
- Pages: Login, Register, Dashboard, Apply Leave, Admin Panel

## ▶️ How to Run
1. Clone the repository  
2. Run `npm install`  
3. Run `npm start`  
4. Open http://localhost:3000
